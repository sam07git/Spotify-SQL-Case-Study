\# 🎧 Spotify User Behavior Analysis (SQL Case Study)



\## 📌 Project Overview



This project analyzes Spotify user behavior using SQL.

The goal is to understand user engagement, subscription patterns, and listening habits.



\---



\## 📊 Dataset



\* 50,000 rows

\* 18 columns

\* Includes data on users, subscriptions, listening hours, skips, devices, and genres



\---



\## 🎯 Key Objectives



\* Analyze user engagement

\* Compare Free vs Premium users

\* Study age group behavior

\* Identify top-performing segments



\---



\## 🔍 Key Insights



\* Free subscription has the highest number of users

\* Free users show slightly higher listening hours

\* No strong factor (skips, playlists, ratings) explains inactivity

\* Age groups show similar engagement patterns

\* 36–50 age group is the largest user segment

\* Skipping behavior is not strongly linked to inactivity



\---
